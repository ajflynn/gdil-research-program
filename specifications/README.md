# Specifications

This directory contains the formal specifications developed and maintained as part of the GDIL Research Program.

These specifications define the conceptual, structural, experimental, and technical artifacts used to investigate Goal-Directed Intelligence Layers (GDILs), goal-directed knowledge application, computable knowledge layers, and related scientific infrastructure.

## Specification Descriptions

### Canonical CBKB-FDO Corpus Design Specification

Defines the structure, composition, and design requirements for the canonical corpus of Computable Biomedical Knowledge-bearing FAIR Digital Objects (CBKB-FDOs) used in GDIL experiments.

### ✓ Canonical GB-FDO Corpus Design Specification

Defines the structure, composition, and design requirements for the canonical corpus of Goal-bearing FAIR Digital Objects (GB-FDOs) used in GDIL experiments.

### ✓ Canonical Item Specification

Defines the Item as a referenceable specification of a computationally meaningful data resource and establishes the foundational model for Items used throughout the GDIL Research Program.

### Experimental Item Assignment Strategy and Protocol

Defines the strategy and protocol used to assign Items to experimental GB-FDOs and CBKB-FDOs within canonical corpora and experimental studies.

### ✓ Experimental Item Space Specification

Defines the construction, composition, and intended properties of the experimental Item Space used in GDIL experiments.

### ✓ GARI v1.0 Implementation Specification

Provides implementation guidance and software requirements for constructing the Goal Alignment Research Instrument (GARI) version 1.0.

### ✓ GARI v1.0 Reference Data Schemas

Defines the reference schemas, data structures, and exchange formats used by GARI v1.0 and related experimental artifacts.

### ✓ GARI v1.0 Specification

Defines the architecture, concepts, workflows, components, and behavior of the Goal Alignment Research Instrument (GARI) version 1.0.

## Notes

Specifications marked with a checkmark (✓) are publication candidates or are otherwise considered mature baseline specifications for the current phase of the GDIL Research Program.

Specifications without a checkmark remain under active development.
